Tuga++ é um compilador de pseudocódigo. Foi desenvolvido com vista ajudar os demais que se estão a iniciar no mundo da programação.

1. Pré-requisitos

 - MinGW (gcc) ( https://sourceforge.net/projects/mingw/files/Installer/mingw-get-setup.exe/download )

2. Como executar?

    1. Depois de instalar o MinGW é necessário adicionar o seu directório bin à variável de ambiente PATH. Se o MinGW tiver sido instalado em C:\MinGW então a PATH deverá ser: $PATH=$PATH+"C:\MinGW\bin;"
    2. Para compilar deverá aceder ao directório do compilador e usando o ficheiro de exemplo executar: compilar Npar.txt Npar
    3. O ficheiro executável Npar.exe será compilado.
